meditation User Story

# Login & Registration
- As a user, I want to register with username, email, and password so that I can create an account.
- As a user, I want to log in using email and password so that I can access my account.
- As a user, I want to see error messages if login or signup fails.
- As a system, I want to store user data in local storage.

# Home Screen
- As a user, I want to see my progress on the home screen so that I can stay motivated.
- As a new user, I want to see an introduction guide so that I can learn how to use the app.
- As a user, I want quick access to frequently used features.

# Detail Screen
- As a user, I want to see detailed information about an item.
- As a user, I want to save or share items.
- As a user, I want to view related items.

# Persistent Data
- As a user, I want my login to persist across sessions.
- As a user, I want my preferences to be saved.
- As an admin, I want activity logs to persist.

# External API
- As a user, I want to see real-time weather updates.
- As a user, I want to see currency conversion.
- As a user, I want to see nearby locations on a map.

# Settings Menu
- As a user, I want to access settings from any screen.
- As a user, I want categorized settings.
- As an admin, I want to control certain settings.

#: Settings Screen
- As a user, I want to enable dark mode.
- As a user, I want to manage notifications.
- As a user, I want to update account details.

# Notifications
- As a user, I want daily reminders.
- As a user, I want feature update notifications.
- As a user, I want to disable promotional notifications.
- As an admin, I want to send targeted notifications.
